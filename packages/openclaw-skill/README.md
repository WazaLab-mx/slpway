# San Luis Way — OpenClaw Skill

Query San Luis Potosí data from any OpenClaw-compatible agent.

## Install

Available on [ClawHub](https://clawhub.ai).

## Commands

- `events` — List upcoming events
- `places` — List places (restaurants, cafés, museums, bars)
- `search "<query>"` — Search across all content
- `weather` — Current weather
- `guides` — Practical visitor guides
- `outdoor` — Outdoor activities
- `culture` — Cultural sites
- `news` — Local headlines
- `exchange-rates` — MXN exchange rates
